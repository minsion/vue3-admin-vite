## 简介

Vue3 Admin Vite 是一个免费开源的中后台管理系统基础解决方案，基于 Vue3、TypeScript、Element Plus、Pinia 和 Vite 等主流技术

## 开发

```bash
# 安装依赖
yarn i

# 启动服务
yarn dev
```
